Python Script to download Btech Degree Examination Results of all students of all colleges
under MG university,Kottayam,Kerala.

The script scraps results of students and saves it in csv format.
It also calculates the GPA of each student, which is not mentioned in the mark list.

Usage:

Run the program and proceed as per the instructions given in the menu.

*Currently works for S6,S4 and S1S2 results announced in December,2013
